Mixins

@mixin heading-font { font-family: sans-serif; font-weight: bold; }
h1 { @include heading-font; }
with parameters
@mixin font-size($n) { font-size: $n * 1.2em; }
body { @include font-size(2); }
with default values
@mixin pad($n: 10px) { padding: $n; }
body { @include pad(15px); }
with a default variable
$default-padding: 10px;
@mixin pad($n: $default-padding) { padding: $n; }
body { @include pad(15px); }
for media
@mixin maxMedia($width) { @media(max-width:#{$width}) { @content; } }
@include maxMedia(1280px) {.main-menu li { padding: 0 20px; } }
logo { @include maxMedia(1430px) { max-width: 250px; }

Javascript cheatsheet

Arrays

unsift();
adds elements at the begginging of an array
push();
adds elements at the end of an array
shift();
removes an element from the beginning of an array
pop();
removes an element from the end of an array
splice();
returns new modified array
splice(indexOfArray, numOfElementsToRmove);
let array = ['today', 'was', 'not', 'so', 'great'];
array.splice(2, 2);
// ['today', 'was', 'great']
removed elements can be stored to a variable:
let newArray = array.splice(3, 2); // ['really', 'happy']
adding elements
splice(indexOfArray, numOfElementsToRmove, addElementInPlaceOfRemoved);
const numbers = [10, 11, 12, 12, 15];
numbers.splice(3, 1, 13, 14); // returns [ 10, 11, 12, 13, 14, 15 ]
slice();
returns new array with extracted elements
slice(indexOfExtraction, endOfExtraction(-1));
let weatherConditions = ['rain', 'snow', 'sleet', 'hail', 'clear'];
let todaysWeather = weatherConditions.slice(1, 3); // todaysWeather equals ['snow', 'sleet'];

npm cheatsheet

installing npm (download node.js first)
npm install
creating package.json - file that contains information about the project
npm init
creating webpack - dev tool
npm install webpack --save-dev
installing jquery
npm install jquery --save
note* to share the project with other devs, you do not need node_moduls folder that holds all the project dependencies, you only need package.json and then by running npm install in the project, the project will automatically be updated based on package.json
uninstalling dependency
npm uninstall jquery --save
installing webpack cli (conmmand line interface)
npm install webpack-cli --save-dev
starting npm scripts - that are included in package.json
npm run dev
installing webpack dev server
npm install --save-dev webpack-dev-server
installing html webpack plugin
npm install html-webpack-plugin --save-dev
installing axios - an http request
npm install axios --save
